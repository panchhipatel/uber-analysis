# 🚖 Uber Ride Analytics Dashboard

A comprehensive **Streamlit-based Uber Ride Analytics Dashboard** that transforms raw ride booking data into meaningful business insights through interactive visualizations, KPIs, and an intelligent data assistant.

## 🌐 Live Demo

🔗 **Demo:** https://uberpy-r57uokt8jbdwkmmzvsidst.streamlit.app/



---

## 📌 Project Overview

The Uber Ride Analytics Dashboard is an interactive data analytics application built using **Python**, **Streamlit**, **Pandas**, and **Plotly**. It enables users to explore Uber ride booking data, analyze operational performance, monitor key business metrics, and visualize trends through an intuitive dashboard.

The application provides multiple analytical modules, including dataset exploration, executive business overview, ride analytics, AI-powered query assistance, and an interactive dashboard for operational monitoring.

---

## ✨ Features

### 📂 Dataset Explorer
- View complete dataset
- Search records instantly
- Filter data by any column
- Display selected columns
- Dataset statistics
- Missing value analysis
- Download filtered dataset

### 📈 Executive Overview
- Gross Booking Value
- Fulfillment Rate
- Average Ride Distance
- Customer Rating
- Business Unit Performance
- Revenue Share Analysis
- Operational Efficiency (CTAT & VTAT)
- Cancellation Audit
- Payment Method Analysis
- Data Quality Report

### 📊 Ride Analytics
Interactive visualizations including:
- Sunburst Chart
- Treemap
- Box Plot
- Sankey Diagram
- Revenue Distribution
- Customer Rating Analysis
- Ride Flow Analysis

### 🤖 Data Assistant
Ask simple questions such as:
- Total rides
- Revenue analysis
- Vehicle type analysis
- Payment methods
- Ride cancellations
- Customer ratings
- Ride distance analysis

The assistant automatically generates charts based on user queries.

### 📉 Interactive Dashboard
- Vehicle-wise bookings
- Revenue by vehicle type
- Ride status distribution
- Payment method distribution
- Ride Distance vs Booking Value
- Customer Rating Histogram
- Cancellation Reason Analysis
- Average Distance by Vehicle Type
- Booking Value Distribution
- Operational Efficiency (CTAT vs VTAT)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/uber-analysis.git
```

Move into the project folder

```bash
cd uber-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
